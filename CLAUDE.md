# 技術棧
- 地球渲染使用 react-globe.gl
- Tailwind CSS
- 後端資料庫連結 supabase

# 專案特色
- 旅行紀錄器
- 網頁是一顆地球，上面可以存許多在許多國家上的標記點（一個國家可以存多個）
- 點擊標記可以查看該地的文字紀錄，照片，影音，旅遊時間
- 後台可以透過地圖精準新增標記地點，並撰寫文字，上傳圖片影音
- 後台需要登入才有權限修改
- 標題為 旅行紀錄，footer 為 © 2026 · fijjj
- 網頁logo為favicon.svg
- 網頁支援亮暗切換，使用正方形按鈕，按鈕只須太陽和月亮圖案，邊框的四角用平滑修飾
- 地球可以選擇轉動模式或是手動拉模式
- 影音或照片在切換時可以直接在照片中有切換上一張或下一張的按鍵
